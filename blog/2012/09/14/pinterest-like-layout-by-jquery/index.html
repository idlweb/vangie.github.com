几个瀑布流风格布局的jQuery插件

Pinterest的瀑布流布局,让人眼睛一亮,看图的过程变得很享受.巧妙的改善了分页让阅读中断的问题,页面懒装载的设计也很好的解决了一次性装载图片过多页面加载缓慢的问题.国内的花瓣网,借鉴了瀑布流的设计,做得挺不错的.

找了几个瀑布流的插件

  • jQuery Masonry
  • jQuery Isotope
  • The Wookmark jQuery plugin
  • jQuery Waypoints plugin 这款插件可以实现无线滚动条
